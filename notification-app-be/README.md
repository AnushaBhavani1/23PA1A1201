# Notification Application Backend

## Description

A backend application for processing notifications with support for priority handling and reusable logging.

## Features

* Notification processing
* Priority handling
* Logging middleware integration
* API communication

## Technologies Used

* Node.js
* Axios
* Dotenv

## Installation

```bash
npm install
```

## Run

```bash
node index.js
```
