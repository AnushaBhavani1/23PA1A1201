# Vehicle Scheduler Backend

## Description

A Node.js backend application that retrieves vehicle and depot data, processes scheduling logic, and uses a reusable logging middleware to record important application events.

## Features

* API integration
* Vehicle scheduling logic
* Logging middleware integration
* Environment variable support

## Technologies Used

* Node.js
* Axios
* Dotenv

## Installation

```bash
npm install
```

## Run

```bash
node index.js
```
